Solana Prediction Market Smart Contract

This project is an open-source Solana prediction market smart contract built using Rust and the Solana program framework.

Features

  • On-chain market creation
  • Decentralized betting system
  • Automated payout distribution
  • Low transaction fees on Solana

Technology Stack
